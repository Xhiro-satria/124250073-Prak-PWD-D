# Deskripsi singkat program
CareNow merupakan web yang menyediakan layanan reservasi dokter secara online, di mana pasien dapat melakukan registrasi dengan mengisi form yang tersedia untuk melakukan pemesanan jadwal pemeriksaan. Melalui platform ini, pasien dapat memasukkan data diri seperti nama, umur, jenis kelamin, jenis layanan, dokter spesialis  beserta jadwal konsultasi yang tersedia. Selain itu Program ini juga dapat menghitung harga layanan(total harga, diskon, admin) bedasarkan umur dan layanan yang dipilih
